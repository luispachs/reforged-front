# NAGO REFORGED - FRONTEND version 0.0.1
ERP para la gestión de productos, invetarios, empleados , nomina y ventas; Ademas, permite la generación de ordenes de producción
ordenes de compra, ordenes de servicios y ordenes de entrega de producto a clientas.
En este solo se manejara la parte con la que interactua los usuarios.

## Project Structure
- `node_module\` - Contiene todos los modulos a usar para el desarrollo de la aplicación.
- `package.json` - Contiene la configuración y el listado de los modulos a usar
- `sha\` - contiene los componentes pre definidos de shadn
- `public\` - Aquí se almacena los archivos publicos de proyecto
- `test\*` - Se almacenas las pruebas a realizar para el proyecto

## Project Info
- Se una ReactJS para este desarrollo.
- Se usa libreria SHADN como libreria de componente.
- Se una Cypress para pruebas E2E y de componentes.
- Al igual que vitest para pruebas unitarias.
